\doxysubsection{Description détaillée}
La classe \mbox{\hyperlink{class_date}{Date}} permet de stocker des dates dans des structures tm. 

La classe \mbox{\hyperlink{class_date}{Date}} permet \begin{DoxyItemize}
\item de stocker une date, par défaut la date du système; \item de récuperer séparement le jour, le mois et l\textquotesingle{}année; \item de récuperer la date sous forme de string; \end{DoxyItemize}
