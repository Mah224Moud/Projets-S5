Description détaillée

Cette classe permet de gérer une liste d'interaction ainsi que les taches (todo) liées à chaque intéraction.

Elle permet notamment l'ajout d'une interaction, d'un todo, leur suppression et l'affichage de toutes les intéractions d'un contact

void GestionInteraction::addInteraction ( const Interaction &  interact)

Ajouter une interaction.

Ajouter une interaction à la liste des intéractions.

Avant d'inserer l'intéraction dans la liste, on la parcourt pour trouver une intéraction avec une date supérieure. L'intéraction est placée dans la liste juste avant celle-ci, si aucune date supérieure n'a été trouvé, elle est placé en fin de liste. De cette façon, on a toujours une liste d'intéraction triée par ordre croissant des dates.
